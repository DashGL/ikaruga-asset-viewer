## Japanese Text

天内
AMAMAI

「積想の里」へ送られて来る前は、凄腕の飛葦鬼設計技術者であった。里へ来た当初は生きる気力を失い、生きる屍状態であったが、風守考人の説得により「積想」製作に加担し、設計から組み立ての大部分をこなしている。

技術屋特有の少々ひねくれた部分があり、人に対してなかなか素直な態度をとれないが、心根は優しい人物である。
